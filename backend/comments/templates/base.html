This is a simple application that performs sentiment analysis on the comments of a youtube video. It can perform the analysis using AI which relies on a transformer model from hugging face, or by using a non AI "bag of words" approach using VADER analysis. The sentiment score is a number between -1 to +1. -1 signifies a strongly negative sentiment and +1 signifies a strongly positive sentiment. The results of the analysis will include some interactive graphs which can be manipulated. For optimal display of the application, select a video with a sufficient number of comments. Enjoy!
